# Bankruptcy_Prediction
Bankruptcy Prediction using the Polish Bankruptcy Dataset from UCI Machine Learning Repository.
The dataset is about bankruptcy prediction of Polish companies.
The bankrupt companies were analyzed in the period 2000-2012, while the still operating companies were evaluated from 2007 to 2013.
You can download the data here [UCI Machine Learning Repository](http://archive.ics.uci.edu/ml/datasets/Polish+companies+bankruptcy+data)
